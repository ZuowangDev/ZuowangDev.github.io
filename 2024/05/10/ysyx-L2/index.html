U2 Shell工具和脚本

bash中的赋值语法：foo=bar（不能加空格）
bash中访问变量语法（指针）：$foo

bash中’’中的字符会直接当做字符串，而””中的会被转义执行

bash中也允许创建函数：
mcd () {
mkdir -p “$1”
cd “$1”
}
-p允许创建多级文件夹
$1表示调用第一个接受的参数
可以将bash函数的参数列表作为字典：$0：脚本名
$1~$9：第几个参数，$@：所有参数，$#：参数个数，$?：前一个命令的返回值，$$：当前脚本的进程识别码，!!：上一条完整的命令，$_：上一条命令的最后一个参数

bash使用STDOUT返回输出值，STDERR返回错误和错误码，退出码可以搭配&&（与操作符）和||（或操作符）使用。

&& , ||：用于连接两个命令
&&：只有当第一个命令成功后（返回0）才会执行第二个命令，||：只有第一个命令执行失败（返回1）才会执行第二个命令
false：返回1；true：返回0；同一行多个命令可以用;分离
