Chapter 2 -- Type, Operators, and Expression

发表于2024-10-20|更新于2024-11-01|The C Programming Language
|浏览量:

Variables and constants are the basic data objects manipulate in a program

There are now signed and unsigned forms of all integer types.

Variable Names

Names are made up of letters, digits; the first character must be a letter. The underscore “_“ counts as a letter; it is sometimes useful forimproving the readability of long variable name.

Data Types and Sizes

  • char: a single byte
  • int: integer
  • float: single precision floating point
  • double: double precision floating point

number of quaifiers: short(16 bit), long(32 bit)

Constants

int; double; float; character; enumeratiion

Enumerations provide a convenient way to associate constant values with names, and compliers need not check that what you stare in such a variable is a valid value for the enumeration.

Declarations

All variables must be declared before use, although certain declarations can be made implicitly by context.

Arithmetic Operators

  • / + - %

Relational and Logical Operators

= > <= <
== !=
