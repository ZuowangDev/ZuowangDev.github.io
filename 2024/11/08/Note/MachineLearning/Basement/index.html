What is Machine Learning

From confusing content to ordered information.

Critical Terminology

Expert system

机器学习的主要任务就是分类

  • 特征
  • 目标变量
  • 训练数据
  • 测试数据

Main Task

监督学习: 分类和回归 无监督学习: 聚类和密度估计
k-近邻算法 线形回归 K-means DBSCAN
朴素贝叶斯算法 最大期望算法
局部加权线性回归 Parzen窗设计
支持向量机 Ridge回归
决策树 Lasso系数估计

Choose

要考虑如下两个问题：

  • 使用机器学习的目的
  • 需要分析和收集的数据是什么

一般不存在最好的算法或者可以给出最好结果的算法。

开发机器学习应用的步骤 $\Delta$

  1. 收集数据
  2. 准备输入数据
  3. 分析输入数据
  4. 训练算法
  5. 测试算法
  6. 使用算法

Python

  1. 语法清晰
  2. 易于操作纯文本文件
  3. 使用广泛
