Essential C++ Chapter 2

  • 函数定义

  • 函数内联，重载，模版
  • 函数指针，指针函数，函数指针数组

编写函数

返回类型，函数名，参数列表，函数体

函数必须被声明，然后才能被调用：需要包括返回类型，函数名，参数列表
exit(-1);(#include)

调用函数

调用函数时，会在内存中建立起程序堆栈(program stack)。

可以降低复制大型对象的负担，一般不使用，主要用于class object

动态内存管理

dynamic extent: 由程序员自行管理(heap memory)，通过new和delete创建和释放

默认参数

局部静态对象

局部静态变量与全局变量都存放在全局数据区，但局部静态变量只能在调用函数时访问。

inline函数

内联函数：

  1. 适合简单的函数体；
  2. 增大代码内存占用，不适合用于递归和大循环的函数体；

函数重载

Overload Function
编译器会根据函数声明中参数列表中的数据类型确定使用的函数体。

模板函数

类似于装饰器

函数指针

头文件

为了使用同一个函数时不进行多次声明，只需要在头文件中声明，对头文件进行include
函数定义不需要加入头文件。多个文件对头文件引用时会出现多个函数定义
inline函数定义需要加入头文件
全局变量需要在头文件中声明
<>和””搜索目录
变量名的声明：extern，使用户可以在外部调用
